# Задание: Реализация алгоритма сортировки

## Описание
Реализуйте функцию для сортировки массива чисел в порядке возрастания.

## Требования
1. Функция должна принимать на вход список чисел
2. Функция должна возвращать новый отсортированный список
3. Исходный список не должен изменяться
4. Можно использовать любой алгоритм сортировки

## Сигнатура функции
```python
def sort_array(arr: List[int]) -> List[int]:
    pass
```

## Примеры

```python

# Пример 1
вход: [3, 1, 4, 1, 5, 9, 2, 6]
выход: [1, 1, 2, 3, 4, 5, 6, 9]

# Пример 2
вход: []
выход: []

# Пример 3
вход: [42]
выход: [42]

```

## Ограничения

    Длина массива: 0 ≤ n ≤ 10^5

    Значения элементов: -10^9 ≤ arr[i] ≤ 10^9
