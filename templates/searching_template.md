## Теория

Бинарный поиск работает за O(log n) времени, где n - размер массива.
Алгоритм:

    Установить левую границу l = 0, правую границу r = n-1

    Пока l <= r:

        Вычислить средний индекс mid = (l + r) // 2

        Если arr[mid] == target, вернуть mid

        Если arr[mid] < target, l = mid + 1

        Иначе r = mid - 1

    Вернуть -1
